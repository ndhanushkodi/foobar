# foobar

Terraform configuration for the `foobar` HCP Terraform workspace.

## Getting Started

1. Add your provider configuration and resources to `main.tf` (or additional `.tf` files).
2. Push changes to `main` to trigger a Terraform run in the [foobar workspace](https://app.terraform.io/app/ndhanushkodi/workspaces/foobar).

## Structure

| File      | Purpose                          |
|-----------|----------------------------------|
| main.tf   | Root Terraform configuration     |
